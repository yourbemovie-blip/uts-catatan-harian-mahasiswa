# Aplikasi Catatan Harian Mahasiswa

Aplikasi ini dibuat untuk memenuhi tugas Ujian Tengah Semester mata kuliah Pemrograman Mobile menggunakan Flutter.

## Judul Aplikasi
Catatan Harian Mahasiswa

## Deskripsi
Aplikasi Catatan Harian Mahasiswa adalah aplikasi sederhana yang digunakan untuk mencatat kegiatan harian mahasiswa. Aplikasi ini dapat digunakan untuk menambah, mengedit, menghapus, dan melihat catatan harian.

## Fitur Aplikasi
- Login pengguna
- Menampilkan dashboard
- Menambahkan catatan baru
- Mengedit catatan
- Menghapus catatan
- Download catatan
- Menampilkan profil mahasiswa

## Teknologi yang Digunakan
- Flutter
- Dart
